# Example OAuth2 with Spring Boot 2

## Config security

### 1. Add dependencies

```
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-security-oauth2</artifactId>
</dependency>
```
